ROCm Platform Core Packages

Binary Version of ROCm Platform

You can perform a simplified ROCm install using our repository server. The package installs the driver, run time, HCC compiler and HIP language run time, as well as test apps to get you ready to start development in about 5–10 minutes.

  • https://github.com/RadeonOpenCompute/ROCm

  • ROCk: https://github.com/RadeonOpenCompute/ROCK-Kernel-Driver
  • ROCt: https://github.com/RadeonOpenCompute/ROCT-Thunk-Interface
  • ROCr: https://github.com/RadeonOpenCompute/ROCR-Runtime
Peer-to-Peer RDMA Source
  • ROCnRDMA: https://github.com/RadeonOpenCompute/ROCnRDMA
  • RDMA performance test: https://github.com/RadeonOpenCompute/rdma-perftest
